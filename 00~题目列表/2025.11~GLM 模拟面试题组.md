# 2025.11~GLM 模拟面试题组

# 技术开发

### **一、JavaScript 核心与深度**

1.  **作用域与执行上下文**：

    - Q: 解释 JavaScript 的作用域链和执行上下文的区别。
    - 追问: 词法作用域和 `this` 的动态绑定有什么关系？
    - 追问: 立即执行函数表达式 (IIFE) 的主要作用是什么？在现代 ES6+ 模块系统中，它还有用吗？

2.  **闭包**：

    - Q: 什么是闭包？它在 V8 引擎中是如何实现内存管理的？
    - 追问: 请举一个闭包在实际项目中可能导致内存泄漏的场景，并说明如何解决。
    - 追问: 循环引用和闭包导致的内存泄漏有何不同？

3.  **原型与继承**：

    - Q: `Object.create(null)` 和 `{}` 创建的对象有什么区别？
    - 追问: 如何在不使用 `class` 和 `extends` 的情况下，实现多重继承的效果？
    - 追问: 什么是 `mixin` 模式？它在 JavaScript 中是如何实现的？

4.  **this 指向**：

    - Q: 箭头函数的 `this` 是如何确定的？为什么它不能用 `call` 或 `apply` 改变？
    - 追问: 请描述 `new` 操作符改变 `this` 指向的完整过程。
    - 追问: 在一个对象的嵌套方法中，如何确保内层方法的 `this` 仍然指向该对象？

5.  **异步与事件循环**：

    - Q: 描述 `Promise` 的三种状态和状态转换规则。
    - 追问: `Promise.then` 的回调函数是同步还是异步执行的？如果是异步，它属于宏任务还是微任务？
    - 追问: 如何实现一个带超时功能的 `Promise`？

6.  **ES 新特性**：

    - Q: `BigInt` 解决了什么问题？它与普通 Number 类型混合运算时会发生什么？
    - 追问: `Atomics` 和 `SharedArrayBuffer` 是用来做什么的？它们在多线程编程中扮演什么角色？
    - 追问: 什么是 `for...of` 循环？它和 `for...in` 以及 `forEach()` 的根本区别是什么？

7.  **内存管理**：

    - Q: V8 的垃圾回收机制中，新生代和老生代的回收策略分别是什么？
    - 追问: 什么是内存碎片？Mark-Compact 算法是如何解决内存碎片的？
    - 追问: 在开发中，有哪些工具或方法可以帮助你分析和定位内存泄漏？
